# RAG App

Generated by Aakaar on its golden webapp scaffold (FastAPI + React/Vite + PostgreSQL).

## Run the tests (no setup needed — hermetic)
```bash
pip install -r requirements.txt
TESTING=1 pytest tests/ -q        # Windows PowerShell: $env:TESTING="1"; pytest tests/ -q
```

## Run locally
1. `cp .env.example .env` and fill in `DATABASE_URL` (PostgreSQL) and `JWT_SECRET`.
2. Backend: `pip install -r requirements.txt && uvicorn backend.main:app --port 8000`
3. Frontend: `cd frontend && npm install && npm run dev` → http://localhost:5173

## Or with Docker
```bash
docker compose up --build
```

## Layout
- `backend/routers/` — API routers, auto-mounted at `/api` (a file exposing `router` is live).
- `database/models.py`, `backend/schemas.py`, `frontend/src/types.ts` — stamped from ONE
  contract; edit the contract, not these, to change shapes.
- `tests/` — acceptance tests; CI-gate green before every ship.
